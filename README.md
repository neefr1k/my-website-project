# my-website-project

. НАЗВАНИЕ ПРОЕКТА
«Горячая штучка»
2. ДЛЯ КОГО ЭТОТ САЙТ
Сайт для тех, кто любит готовить дома -от начинающих до опытных хозяек и хозяев.
Целевая аудитория: люди, которые ищут простые, понятные и проверенные рецепты без сложных ингредиентов.
Также сайт будет полезен тем, кто хочет делиться своими рецептами и обсуждать кулинарные идеи.
3. КАКИЕ СТРАНИЦЫ БУДУТ НА САЙТЕ 
    1. Главная
- приветствие, кратко о блоге.
- список последних добавленных рецептов (картинка + название + краткое описание).
- ссылки на популярные рецепты.
- поиск по сайту.
    2. Каталог рецептов
- все рецепты с фильтрацией по категориям (завтраки, супы, второе, выпечка, салаты).
- возможность сортировки по дате или популярности.
- каждый рецепт представлен миниатюрой (фото, название, время приготовления).
    3. Страница отдельного рецепта
- фото блюда.
- список ингредиентов.
- пошаговый способ приготовления.
- кнопка «Сохранить в избранное» (для зарегистрированных).
- блок комментариев с возможностью оставить отзыв или вопрос.
    4. О блоге / О нас
- рассказ о том, кто ведёт блог и почему делится рецептами.
- контакты для связи.
- предложение сотрудничества (для брендов / других блогеров).
    5. Блог / Статьи
- заметки о кухонной технике, полезных советах, продуктах.
-возможность комментировать статьи.
    6. Контакты
-форма обратной связи (имя, email, сообщение).
- ссылки на соцсети (Telegram, Instagram, VK).
-карта (опционально - схема, если будет офлайн-проект).
Минимум 5 страниц выполнено. На практике страниц может быть больше за счёт динамического каталога.
4. ЧТО УМЕЕТ ДЕЛАТЬ САЙТ
-Показывать рецепты и статьи в удобном для чтения виде.
-Искать рецепты по названию или ингредиентам.
-Фильтровать рецепты по категориям (например: «супы», «выпечка»).
-Регистрировать новых пользователей.
-Авторизовать пользователей (вход по email + пароль).
-Давать возможность зарегистрированным пользователям:
-оставлять комментарии под рецептами;
-добавлять рецепты в избранное;
-предлагать свой рецепт на публикацию (через админа).
-Принимать заявки / сообщения через форму на странице «Контакты».
-Администратору:
-добавлять / редактировать / удалять рецепты и статьи;
-управлять пользователями (блокировка, удаление);
-модерировать комментарии.
5. ТЕХНОЛОГИИ
-LAMP (Linux, Apache, MySQL, PHP) -серверная среда для работы сайта.
-WordPress - система управления содержимым (проще и быстрее для кулинарного блога).
-GitHub -хранение кода (если нужна будет разработка) или документации проекта (ТЗ, схемы).
-Draw.io -для создания диаграмм (Use Case, Deployment).
